## Status Light

* The crossing gate system includes a status light located on the controller enclosure.

* Solid red light: Normal operation.

* Blinking red light: Error state.

* No Light: System is off, no power.

If in an error state, use a voltmeter to check the battery voltage level. If needed, recharge the batteries using a smart 24V charger.
